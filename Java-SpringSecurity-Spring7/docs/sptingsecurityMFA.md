springsecurity7においてMFA（多要素認証）が簡単に実装することが可能になったため忘備録として残しておきます。


springsecurity7では認証の際に、確認証メカニズムが対応する権限を自動的にAuthenticationに追加するようになりました。

認可ルールであらかじめ認証他所を記述しておくと要素権限が不足した際にSpringSecurityがその権限を取得できるエンドポイントに自動的にユーザーをリダイレクトしてくれます。これによってリダイレクトロジックを自前で書くことが必要なくなりました。（書いていた時を知らない、、、）

⭐️実装方法

今回は簡単に実装したいため@EnableMultiFactorAuthenticationアノテーションを使用していきます
