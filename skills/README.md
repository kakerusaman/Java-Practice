# skills

学習記録をまとめるフォルダです。

---

## フォルダ構成

```
skills/
├── README.md         # このファイル（説明・ルール定義）
├── template/
│   └── TEMPLATE.md  # 学習記録のテンプレート
└── records/          # 学習記録ファイル置き場
```

---

## ルール

1. **ファイル名は英語**（例: `SpringSecurity.md`、`MyBatis.md`）
2. **1ファイル1技術テーマ**（テーマが大きい場合はサブテーマで分割可）
3. **コードは必ずコードブロックで記載**
   ````
   ```java
   // コードはここに書く
   ```
   ````
4. **参考資料は必ず記載**（公式ドキュメントのURLなど）
5. **テンプレートを使用する**（`template/TEMPLATE.md` をコピーして使う）
6. **つまずいたポイントと解決方法は必ず記載**（同じミスを繰り返さないため）
